Welcome to our Animated Algorithms site.

The animations are created by the Teaching Machine, which is a computer model of a computer. The Teaching Machine interprets Java or C++ code and presents visualizations of the state of the computer as it executes the code. In addition to standard visualizations, annotations added to the source the code can be used to create custom --program defined-- visualizations; this facility was used to create the animations you will see at this site. The web pages were created with the help of WebWriter++.

This site is intended for students taking courses in algorithms. Instructors interested in contributing additional examples should contact the authors. Companion sites provide information on the Teaching Machine, WebWriter++, and Program Defined Visualizations.
