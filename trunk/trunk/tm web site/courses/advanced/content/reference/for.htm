Syntax

for (initial expression; test expression; increment expression )

   statement

where statement is any single statement, including an if statement or a loop

for (initial expression; test condition; increment expression) {

   statement1

   statement2

   ...

}

where any number of statements may be included in the block (including just one, making it equivalent to the first form).

For both forms, the initial expression is executed just once, first, before anything else. Then the test condition is carried out. If the result of the test is true, the body of the loop (the single statement following the ) in the first form, the block enclosed in { } in the second form) is executed, followed by the increment expression.

The sequence, test condition , execute body, increment expression continues to be executed until the test expression is false.

if the test condition is false the first time it gets executed, the body and increment expression will never be executed.

Examples

for (int i = 0; i < 10; i++)

   cout << "i is " << i << endl;

results in the output

i is 0
i is 1
......

i is 9

The loop

for (char c = 'a'; c <= 'z' ; c++){

     cout << c;

     if (c < 'z') cout << ' ';

}

results in the entire lowercase alphabet being printed out with a single space between letters
