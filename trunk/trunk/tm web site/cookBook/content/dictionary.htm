Higraph and TM Dictionary

Active View Set

The set of Visualizers to which View commands are sent.

Branch

A visible link between a parent node and a child node.

Child or Child Node

A child is a node which has been attached to a parent node.

Datum

The Teaching Machine's internal model for a piece of data. Datums are quite sophisticated. For example a datum knows the name attached to the piece of data (if there is one), its type, value and address. If it is compound(for example, an array) it knows the number of children and can access them all and the children themselves will be datums.

Edge

An edge can be created between any two Nodes in a Higraph. The edge is directed from a source Node to a target Node.

Higraph Model

A Higraph contains 0 or more Nodes, each of which is either a child of another Node or a TopNode. Thus a Higraph can be considered as a forest or trees. Directed Edges may be created from any Higraph Node (the source Node) to any Higraph Node (the target Node)

Higraph View

A particular view of the Higraph Model. It could be of the wholeGraph or a subGraph.

Node

A node is the fundamental component of a Higraph. Either it is assigned as a child of another Node or it is considered a TopNode.

A node to which another node has been assigned as a child is said to be a parent of the child node.

Parent or Parent Node

Any node which has 1 or more child nodes attached to it.

Program DependentVisualization or PDV

A Visualization driven by code running on the Teaching Machine

Scripting Call

A call from within the demonstration code being run on the Teaching Machine. A typical call is of the form

ScriptManager.relay("HigraphManager", "setDefaultNodeValueColor", WHITE);, from inside Java code. (The call would be identical from inside C++ code except that the '.' is replaced by ::

where relay is a standard ScriptManager function that relays the call to the HigraphManager module in the Teaching Machine which has a member function called setDefaultNodeValueColor taking a single int argument.

All TM Higraph calls use the relay mechanism.

Script Manager

A module in the Teaching Machinewhich receives all scripting calls from within the demonstration code being run on the Teaching Machine.

Source or Source Node

A node on which an edge begins.

SubGraph

A graph, with a user defined id, which contains 0 up to all of the Nodes of the Higraph. Nodes in a subgraph have the same relationship to other Nodes of the same subGraph as they do in the Higraph

Target or Target Node

A node on which an edge terminates.

TopNode

Any Node which is not a child of another Node. That is any Node which has no parent node.

Visualizer

A window in the Teaching Machinefor displaying a Higraph View. The Teaching Machine supports multiple visualizers.

WholeGraph

The subGraph which contains all of the Nodes of the Higraph. It has the reserved id "wholeGraph".
