Matrix Tutorial Part I
This tutorial looks at the representation of two-dimensional matrices using C++'s arrays.

The matrix is represented by an array; the number of items in the array is equal to the product of the number of rows and the number of columns.

In this example there are 2 rows and 3 columns, so we need an array of size 6.

We'll use the variables i and j to iterate over the number of rows and columns of the matrix respectively.

In this example, we want to represent a matrix with 2 rows and 3 columns. We will use array A[] to represent this matrix.

We will number rows and columns of the matrix starting with 0. Recall that, in C++, array items are also numbered from 0.

Notice that the representation of row i of the matrix begins at item i*cols of the array. I.e., the representation of row 0 starts at item 0 of the array, and the representation of row 1 begins at item 3 of the array.

Each matrix item (i,j) --i.e., the item in row i and column j-- is represented by item ((i * cols) + j) of the array.

As you'll see, using this formula allows us to traverse the entire matrix by iterating over the rows and columns.

Here, the formula gives us: (i * cols) + j == (0 * 3) + 0 == 0. So item A[0] represents item (0,0), of the matrix.

This loop initializes the matrix to values 10, 11.5, 13, ... .

Item (0,0) of the matrix has been assigned the value of 10.0 and val has been updated to 11.5.

Here, the formula gives us: (i * cols) + j == (0 * 3) + 1 == 1. So item 1 of the array represents item (0,1) of the matrix.

Item (0,1) the matrix has been assigned the value of 11.5 and val has been updated to 13.0.

>Here, the formula gives us: (i * cols) + j == (0 * 3) + 2 == 2. so item 2 of the array represents item (0,2) of the matrix.

Item (0,2) of the matrix has been assigned the value of 13.0 and val has been updated to 14.5.

Here, the formula gives us: (i * cols) + j == (1 * 3) + 0 == 3. so item 3 of the array represents item (1,0) of the matrix.

Item (1, 0) of the matrix has been assigned the value of 14.5 and val has been updated to 16.0.

Here, the formula gives us: (i * cols) + j == (1 * 3) + 1 == 4. So item 4 of the array represents item (1,1) of the matrix.

Item (1,1) the matrix has been assigned the value of 16.0 and val has been updated to 17.5.

Here, the formula gives us: (i * cols) + j == (1 * 3) + 2 == 5. so item 5 of the array represents item (1,2) of the matrix.

The final item of the matrix has been assigned the value of 17.5 and now the initialization is complete.
