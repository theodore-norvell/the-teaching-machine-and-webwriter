Library Declaration

#include <string>

must be typed into any file in which string objects are to be used. The at function can only be called with respect to an object of the string class, using the member-of . operator.

char at(int index );

returns the char at position index of the invoking string object. Specifying an index outside the range [0, length() - 1] will cause a run-time error.

Examples

string hi = "hello world!\n";

cout << hi.at[6];

results in the output w.
