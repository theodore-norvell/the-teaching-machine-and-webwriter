Matrix Tutorial Part I
This tutorial looks at the construction and usage of a two-dimensional matrix.

The matrix is created as an array with it's number of cells equal to the product of the number of rows and the number of columns.

Notice the arbitrary values stored at all the matrix locations when it is declared.

We'll use the variables i and j to iterate over the number of rows and the number of columns in the matrix respectively.

The matrix form of array A[] is a standard 2x3 matrix.

We'll assign the matrix location represented by ((i x number of columns) + j) the cureent value of the variable val and then increase val by 1.5.

As you'll see, using this formula allows us to traverse the entire matrix while iterating over i and j.

The first element of the matrix is assigned the value of 10.0 and val is updated to 11.5.

Work out the locations accessed for the rest of matrix as the loops continue, or skip ahead if you feel you're comfortable with the matrix lookups.
