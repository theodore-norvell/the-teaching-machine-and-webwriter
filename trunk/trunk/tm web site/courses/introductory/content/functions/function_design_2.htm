Problem: Evaluate an integral of a function F(x) given the upper and lower limits using the trapezoidal rule

Function

F(x) = x . sin(x2) - cos(x)

Trapezoidal Rule

Problem Solution

What are the basic steps?

  1. Get the inputs
  2. Evaluate the equation
  3. Output the answer

The only one that needs elaboration is step 2.

Evaluate the equation --->

answer = (b-a) * (F(a) + F(b) )/2
