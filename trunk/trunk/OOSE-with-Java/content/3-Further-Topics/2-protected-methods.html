Then subclasses can access the interest rate, but can not change it.

If we want to allow subclasses to alter the interest rate, we can add a method

    protected void setInterestRate(double interestRate ) {
        assert interestRate >= 0 ;
        this.interestRate = interestRate ; }

Then regardless of the implementation in the subclasses, the invariant is respected.
