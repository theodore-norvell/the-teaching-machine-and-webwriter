Matrix Tutorial Part II

This tutorial looks at passing a two-dimensional matrix to a function as an argument. (Click here to go to part I)

Let's pass the matrix into a function that will scale it using a scale factor we have defined. We also need to tell the function the dimensions of the matrix and the scale factor itself, so we'll pass those in as arguments to the function as well.

The parameter for the array is called B. Within the function we use B as the name for the array. Although it is known by the name B within the scale function and the name A within the main function, there is only one array.

We'll use the lookup formula B[(i*cols)+j] you learned in Part I to identify item (i,j) of the matrix..

Using the formula: (0 * 3) + 0 = 0.

You can iterate through the rest of the matrix cell-by-cell, or skip ahead if you've got the hang of traversing and assigning values to the matrix.

Using the formula: (0 * 3) + 1 = 1.

(0 * 3) + 2 = 2.

(1 * 3) + 0 = 3.

(1 * 3) + 1 = 4.

(1 * 3) + 2 = 5.

We're now at the end of the scale function. It's time to return to the main function.

Back in main, the matrix is refered to by A again. The name B had its scope limited only to the scale function and can no longer be used.

Note how the values of the original matrix A have been modified. The values of the other variables such as i and j, however have not and continue to store the value they did before we entered the function.

Let's pass another matrix into the scale function. This time, we'll make it a 2x2 matrix.

Once again, we'll pass in the matrix (this time, matrix C), its dimensions, and the scale factor.

This time, matrix C is refered to by the name B within the scale function. Let's skip to the end of the function, since you now know how to iterate over a matrix.

Time to return to main.

Back in main, the name B for the matrix is lost again.
