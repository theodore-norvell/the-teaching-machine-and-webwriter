For-loop tutorial

This tutorial explains the working of a for-loop with examples.

Example 1: Checking if a number is prime or not.
It is assumed that the number is greater than 1.The function is called isPrime() which returns boolean variables true or false.

First we declare a bool variable b and assign it to the return value of the isPrime() function. isPrime() returns true if the number passed to it is prime and returns false otherwise. We pass number 7 to the isPrime() function to verify that it is prime .Notice that b is intially false.

Notice that in the expression engine isPrime(7) is evaluated next.

In the isPrime() definition: The number is copied into a local variable called num.

A bool variable called result is declared and initialised to true.

The for loop: In the first iteration a variable div local to the loop is created and intialized to 2.

Then the condition : div < num ? is checked. Yes, 2 is less than 7, so the body of for loop is executed next.

The for loop body: The remainder of num/div (given by num%div) is logically ANDED with result. Here 7%2 = 1. true && (1!=0) becomes true&&true = true So, result = true.

So, if the remainder of any division is 0 i.e. num is divisible by a div(ranging from 1 to num) then result becomes false because (0!=0) is false and false ANDed with any boolean is false.

Next div is incremented. Did you notice the sequence of execution of expressions in the for loop? First initialization ,then condition, then body and after that increment!

Iteration 2: Condition div < num = 3 < 7 is true. So its the body of the for loop next.

div is 3 , num is 7 so num%div is 2 ,therefore result is true&&(1!=0) which is true&&true, which is true.

Iteration 3: div is 4 , num is 7 so num%div is 3 ,therefore result is true&&(3!=0) which is true&&true, which is true.

Iteration 4: div is 5 , num is 7 so num%div is 2 ,therefore result is true&&(2!=0) which is true&&true, which is true.

Iteration 5: div is 6 , num is 7 so num%div is 1 ,therefore result is true&&(1!=0) which is true&&true, which is true.

Now when div is incremented it becomes 6+1=7. The condition div < num becomes false. So instead of the body the control jumps out of the for loop.

The statement immediately following the for loop returns the result to the main function. Notice that now b becomes true.

Example 2: This is a very simple example to illustrate the usage of for loop. The Calcsum() function calculates the sum of first n natural numbers where n is passed as an argument and returns the sum.

The number passed is copied into a local variable n and another local variable result is initialized to 0. Here we pass 6 to the function and intialize the variable result to 0.

The for loop: A loop variable i intialized to 1 the first natural number.

In every iteration the following occurs: i is added to the result. It is incremented by 1. If i < n is still true the new i is added to the result. This happens until i < num condition becomes false.

1 <= 6 is true

result becomes 0+1=1

i++ increments i by 1 and i becomes 1+1=2

2 <= 6 is true

result is 1+2=3

i++ makes i=2+1=3

3 <= 6 is true

result becomes 3+3=6

i++ makes i=3+1=4

4 <= 6 is true

result=6+4=10

i++ makes i=4+1=5

5 <= 6 is true

result becomes 10+5=15

i++ makes i equal to 5+1=6

6 <= 6 is true

result becomes 15+6=21

i++ makes i=6+1=7

7<=6 is false

The condition becomes false so the control comes out of the for loop. The result contains 21 which is now returned to the main function.

In the main function now the variable sum contains the sum of first 6 natural numbers.
