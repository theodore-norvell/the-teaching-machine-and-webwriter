Library Declaration

#include <string>

must be typed into any file in which string objects are to be used. The insert function can only be called with respect to an object of the string class, using the member-of . operator.

void insert( int start, string newString );

inserts newString into the invoking string starting at position start.

Examples

string hi = "hello world!\n";

hi.insert(6,"beautiful ");

cout << hi;

results in the output hello beautiful world!\n .
