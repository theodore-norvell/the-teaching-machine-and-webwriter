Matrix Tutorial Part I
This tutorial looks at the representation and usage of a two-dimensional matrix.

The matrix is represented by an array with its number of items equal to the product of the number of rows and the number of columns.

Notice the arbitrary values stored at all the matrix locations when it is declared.

We'll use the variables i and j to iterate over the number of rows and the number of columns in the matrix respectively.

The matrix represented by array A[] is a 2 row by 3 column matrix. Notice that the representation of row 1 begins at 1*cols. In general the representation of each row begins at item i*cols of the array. The size of the array is 2*3, i.e. 6.

Each matrix item (i,j) is represented by item ((i * number of columns) + j) of the array.

As you'll see, using this formula allows us to traverse the entire matrix by iterating over the rows and columns.

Here, the formula gives us: (0 * 3) + 0 = 0.
