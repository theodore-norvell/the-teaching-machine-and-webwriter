Syntax

if (condition)

   statement

where statement is any single statement, including another if statement or a loop

if (condition){

   statement1

   statement2

   ...

}

where any number of statements may be included in the block (including just one, making it equivalent to the first form).

For both forms, if the condition is true, the single statement (first form) or block (second form) immediately following the if (known as the then clause) is executed. If the condition is false, it is bypassed.

Examples

if (x < 0 )

   cout << "x is negative." << endl;

single cout statement

if (x < 0 )

   for(int i = 0; i < 10; i++)

      cout << "i = " << i << endl;

the for loop counts as a single statement

if (x < 0 ){

   cout << "x is negative"

   if (x < 10)

      cout << " and it's less than 10."

   cout << endl;

}

All three statements in the block following the if are executed when x is negative. The if inside the block and the cout statement that follows it act effectively as a single statement.
