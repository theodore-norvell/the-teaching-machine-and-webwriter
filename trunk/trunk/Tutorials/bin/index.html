This tutorial looks at a data structure for dynamically resizable array.

The structure consists of a size field and an array that is at least as long as the size.

Items of the array past the first 'size' are ignored.

I've initialized size to 0 and the array to length 1.

The 'grow' method may be called to make the array larger by one

In this case the array is bigger than the 'size' field, so we only need to increment the size field.

The grow function is about to be called for the second time.

This time the size is already as big as the array.

We need a bigger array.

Allocate a new array that is twice the length.

Next. Copy the first 'size' items to the new array.

Next. Replace the current array with the new one.

Finally. Increment the size.
